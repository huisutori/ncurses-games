# ncurses-games
Ncurses Games which can be played in Linux or MAC OS.

# How to Play
After cloning this project, type below in your terminal.
```
$ make
$ ./cgames
```
#### NOTE: For the optimal expreience, make sure to increase your terminal screen and font size.
